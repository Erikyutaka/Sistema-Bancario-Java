# Projeto de Sistema Bancário em Java

Projeto desenvolvido para praticar conceitos de programação orientada a objetos.

## Tecnologias
Java

## Funcionalidades
- Criar contas
- Depositar valores
- Realizar saques
- Simular operações bancárias

## Conceitos aplicados
- Programação Orientada a Objetos
- Herança
- Polimorfismo
